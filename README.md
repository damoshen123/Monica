# Node.js中转站服务器

这是一个用于Tampermonkey脚本的Node.js中转站服务器，专为Windows环境设计。

## 快速开始

1. 确保您的系统已安装Node.js和npm。如果没有，请从[Node.js官网](https://nodejs.org)下载并安装。

2. 下载或克隆此仓库到您的本地机器。

3. 双击运行`Start.bat`文件。这将自动安装依赖并启动服务器。

服务器将在 http://localhost:3000 上运行。
